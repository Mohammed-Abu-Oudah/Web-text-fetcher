# Web-text-fetcher
This is a website page constructed with html, css, and javascript which fetches quotes from an api and displayes them.

If there's an internet connection problem or the api is not correct, it displayes an error.

If you click the submit button without typing anything in the input field, then all the quotes are returned as a list.

If you click the submit button after typing a text in the input field:
    If any of the quotes contain this string then only these quotes are returned.
    If None of the quotes contain this string then non of the quotes is returned.
